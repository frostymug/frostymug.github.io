Ice Age Trail App

Interactive educator/natural-history map for the Ice Age Trail. Current stage: intake with deployment spike succeeded.

Status

  • Stage: Intake -> Deployment Spike Succeeded
  • Primary stakeholder: Lisa Siewert, original author of the capstone prototype
  • Current repo: https://github.com/ecolville/GEOG778_Capstone
  • Current deployment evidence: Deployment spike succeeded on 2026-06-19. Lisa's static app can be deployed as a controlled Andy/Aspen prototype; keep it framed as a feasibility/demo deployment, not an official Ice Age Trail Alliance app.
  • Latest stakeholder evidence: Andy and Lisa had a 2026-07-10 check-in. Gemini notes are preserved; a verbatim transcript is still outstanding. The conversation shifted the next architecture question from front-end hosting to data-layer stewardship, migration, and backup.
  • ArcGIS licensing note: Esri's ArcGIS for Personal Use license is limited to personal, noncommercial use and excludes third-party benefit. Esri's current API-key docs also say API key authentication is not available for ArcGIS Online subscriptions purchased through Personal Use, which matters because Lisa's app expects window.ARCGIS_API_KEY.
  • Approval: Draft / Aspen synthesis from Andy-Lisa calls and repo inspection

Working Project Shape

Lisa built a static ArcGIS JavaScript app as a graduate capstone. It currently focuses on an educator geologic map for a limited north-central Wisconsin area, with trail segments, geologic layers, POIs, classroom prompts, and printable activities.

The opportunity is to turn that prototype into a maintainable, expandable project that can eventually support statewide Ice Age Trail coverage, richer points of interest, contributor photos, and broader natural-history content.
